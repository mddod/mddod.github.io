MDOD: 多维数据异常值检测

English

这是一个多维数据异常值检测python库。

MDOD Paper

  • MDOD算法发表在ICAIIC 2024: "Outlier Detect using Vector Cosine Similarity by Adding a Dimension" (论文)

安装

  • pip install mdod

    or

    git clone https://github.com/mddod/mdod.git
    cd mdod
    python setup.py install

使用举例 (数据读取于TestDataset.txt):

  • import numpy as np
    import mdod
    localFile = 'TestDataset.txt'
    dets= np.loadtxt(localFile,delimiter=',')
    # nd: 观察点在新增维度上的值
    nd = 1
    # sn: 统计的数量（计算值从大到小排序）
    sn = 15
    result = mdod.md(dets,nd,sn)
    print(result)
