MDOD: Multi-Dimensional data Outlier Detection, 多维数据异常值检测

( English)


Python library for Multi-Dimensional data Outlier/Anomaly Detection algorithm, 多维数据异常值检测Python库。

MDOD paper

  • MDOD paper is published in ICAIIC 2024 as title "Outlier Detect using Vector Cosine Similarity by Adding a Dimension" (paper)

安装方法：

  • pip install mdod

    或者

    git clone https://github.com/mddod/mdod.git
    cd mdod
    python setup.py install

使用示例：

(testmdod.py)
  • import numpy as np
    import mdod
    localFile = 'TestDataset.txt'
    dets= np.loadtxt(localFile,delimiter=',')
    nd = 1
    sn = 15
    result = mdod.md(dets,nd,sn)
    print (result)
