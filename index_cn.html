MDOD: 多维数据异常值检测

English

这是一个多维数据异常值检测python库。

MDOD Paper

  • MDOD算法发表在ICAIIC 2024: "Outlier Detect using Vector Cosine Similarity by Adding a Dimension" (论文)

安装

  • pip install mdod

    Or via GitHub

    git clone https://github.com/mddod/mdod.git
    cd mdod
    python setup.py install

  • MDOD的参数包括：
    norm_distance：虚拟维度的距离（默认1.0，影响相似度计算）。
    top_n：考虑用于统计的最临近点的数量（选择合适的数量用于异常值识别区分）。
    contamination：设置的异常比例（默认0.1，用于阈值计算）。
    sampling_rate：采样率（0~1，默认1.0，低值加速计算）。
    random_state：随机种子。
